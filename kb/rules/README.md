# AI Agent Rules

This directory stores rules and guidelines for AI agents.

## Format

Each rule file should follow this naming convention:
- `[category]_rules.md` for category-specific rules
- `general_rules.md` for general guidelines

## Content Structure

```markdown
# Rule Category

**Priority:** [High/Medium/Low]
**Applies To:** [Agent types or contexts]

## Rules

1. Rule description
   - Details and examples
   - Exceptions

2. Another rule
   - Details and examples
   - Exceptions

## Examples

[Examples of applying these rules]
```

## Categories

Common rule categories:
- Code style and formatting
- Security and privacy
- Communication guidelines
- Error handling
- Documentation standards
- Testing requirements
